## 📋 项目简介

本项目是一个基于 **Three.js** 的杭州西湖三维数字孪生交互系统。系统以真实西湖地理数据为基础（1单位≈1米），构建了包含西湖核心景区的虚拟三维场景，支持多视角浏览、图层切换、昼夜切换、景点导航、建筑点击详情等丰富交互功能。

系统采用程序化建模 + **GLTF/GLB外部模型加载**的混合方案，支持加载由**腾讯混元3D**等AI生成式工具创建的精细模型资产。

🌐 **在线预览**: 使用任意HTTP服务器在项目根目录启动，浏览器打开 `index.html` 即可。

```bash
# 方式1：Python
python -m http.server 8080

# 方式2：Node.js
npx serve .

# 方式3：VS Code Live Server 插件
```

## 🎯 功能特性

### 核心功能
| 功能 | 描述 |
|------|------|
| 🗺 **三维场景** | 基于真实西湖地形数据（5000m×5000m），包含湖面、苏堤、白堤、孤山、夕照山等地理要素 |
| 🏛 **建筑模型** | 雷峰塔(71.68m)、三潭印月(三塔62m间距)、断桥(28.8m)、曲院风荷、灵隐寺等10+景点 |
| 🌳 **植被系统** | 苏堤/白堤行道树、湖岸散落树、孤山植被，共200+棵树木 |
| 📐 **精确比例** | 基于真实数据：苏堤~2200m、白堤~987m、三潭印月400m到雷峰塔、500m到孤山 |

### 交互控制（≥3类）
| 交互 | 操作 |
|------|------|
| 🖱 **鼠标控制** | 左键旋转视角、右键平移、滚轮缩放 |
| 🎯 **景点导航** | 点击右侧景点列表自动飞抵目标位置 |
| ⌨ **键盘快捷键** | `1-5`图层切换、`D`昼夜、`W`线框、`E`导出GLB、`R`重置、`Esc`关闭详情 |

### 视角模式（≥2种）
| 模式 | 描述 |
|------|------|
| 🎯 自由轨道 | 默认模式，可旋转/平移/缩放 |
| 🕊 飞行鸟瞰 | 从湖面上方俯视全景 |
| 📐 正俯视图 | 纯顶视图，适合观察整体布局 |

### 高级功能（加分项）
| 功能 | 描述 |
|------|------|
| 🌙 **昼夜循环** | 一键切换白天/黑夜，光照/天空/雾效联动变化 |
| 🏗 **5层图层切换** | 建筑/植被/水域/地形/标注 独立显隐 |
| 📐 **线框模式** | 全局线框渲染，便于观察几何结构 |
| 🚣 **动画小船** | 4条小船沿不同路径在湖面航行动画 |
| 🕊 **动态飞鸟** | 25只飞鸟在场景上空盘旋动画 |
| 🖱 **射线拾取** | 点击任意建筑弹出详细图文信息卡片 |
| 💾 **GLB导出** | 浏览器端一键导出当前场景为GLB文件（支持Blender等工具二次编辑） |
| 📦 **GLTF/GLB加载** | 支持加载外部3D模型替换程序化模型，兼容AI生成资产 |

## 📝 开发说明

### 场景比例
场景采用 **1单位≈1米** 的比例尺。为在有限屏幕空间内展示完整的西湖景区，部分建筑的详细尺寸按场景整体比例做了缩放，但**相对位置关系严格遵循**提供的真实地理数据。

### 模型替换
如需使用AI生成的精细模型：
1. 使用腾讯混元3D或其他工具生成GLB格式模型
2. 将文件放置于 `models/` 目录
3. 确保文件名与 `loadGLBModels()` 函数中的路径一致
4. 刷新页面即可看到新模型

### 浏览器兼容
- Chrome 90+ ✓
- Edge 90+ ✓
- Firefox 90+ ✓
- Safari 15+ ✓
- 需要支持 WebGL 2.0
